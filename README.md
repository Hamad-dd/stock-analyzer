# 📊 Smart Asset Analyzer

An advanced multi-asset analysis platform that supports stocks, gold, and oil, combining technical analysis, financial insights, and AI-powered predictions to assist smarter investment decisions.

---

## 🚀 Key Features

- 📈 Multi-asset analysis (Stocks, Gold, Oil)
- 📊 Technical indicators (RSI, MACD, Moving Averages)
- 🧠 AI-based prediction using Machine Learning (Random Forest)
- ⚠️ Risk assessment and decision support
- 🔄 Asset comparison mode
- 📉 Trend and volatility analysis
- 📤 Automated Excel report generation
- 📊 Interactive data visualization dashboard

---

## 🤖 AI Module

This project integrates a machine learning model that analyzes historical price data and technical indicators to predict the possible price direction.

- Model: Random Forest
- Output: Uptrend 📈 / Downtrend 📉
- Confidence Score included

---

## 📦 Supported Assets

- Stocks (e.g., AAPL, TSLA, NVDA)
- Gold
- Oil
